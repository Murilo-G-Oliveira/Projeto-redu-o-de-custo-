# Projeto-redu-o-de-custo-

RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 03/04/2026
Empresa: Abstergo Industries
Responsável: Murilo Oliveira

Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por mim, Murilo Oliveira. O objetivo foi escolher três serviços da AWS que ajudassem a reduzir custos de forma imediata e melhorar a forma como a infraestrutura da empresa funciona.

Descrição do Projeto
O projeto foi dividido em três etapas, cada uma com foco em um serviço específico:
Etapa 1:
- Amazon EC2
- Foco da ferramenta: Substituir servidores físicos por instâncias em nuvem.
- Descrição de caso de uso: Migrei parte dos sistemas internos para EC2, usando instâncias reservadas para cargas críticas e instâncias spot para tarefas menos importantes. Isso trouxe flexibilidade e reduziu bastante os gastos com manutenção de hardware.
Etapa 2:
- Amazon S3
- Foco da ferramenta: Armazenamento seguro e barato para os dados da empresa.
- Descrição de caso de uso: Centralizei os arquivos corporativos em buckets S3, configurei versionamento e políticas de ciclo de vida para mover dados antigos para classes de armazenamento mais econômicas. Com isso, conseguimos cortar custos e garantir maior segurança.
Etapa 3:
- AWS Lambda
- Foco da ferramenta: Automação de processos sem precisar de servidores dedicados.
- Descrição de caso de uso: Criei funções Lambda para processar logs, disparar notificações e gerar relatórios automáticos. Isso eliminou servidores rodando o tempo todo e deixou os processos internos mais ágeis.

Conclusão

A implementação de ferramentas na empresa Abstergo Industries trouxe como resultado redução de custos, maior escalabilidade e automação de tarefas, aumentando a eficiência e a produtividade da equipe. A recomendação é continuar usando esses serviços e avaliar novas soluções da AWS que possam agregar ainda mais valor, como RDS para banco de dados e CloudWatch para monitoramento.
